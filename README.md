# Wazuh SIEM Deployment and Endpoint Monitoring

##  Project Overview

This project demonstrates the deployment and configuration of the Wazuh SIEM platform in a virtual environment. The objective was to deploy the Wazuh Virtual Appliance, configure the Wazuh Manager, install Windows and Linux agents, enable File Integrity Monitoring (FIM), and verify security events through the Wazuh Dashboard.

---

##  Objectives

- Deploy the Wazuh Virtual Appliance
- Configure the Wazuh Manager
- Install Wazuh Agent on Windows and Linux
- Register both agents with the Wazuh Manager
- Configure and validate File Integrity Monitoring (FIM)
- Generate and verify security events

---

##  Technologies Used

- Oracle VirtualBox
- Wazuh SIEM v4.14.x
- Windows 11
- Linux (Ubuntu 24.04 LTS)
- Wazuh Dashboard

---

##  Project Implementation

The following tasks were successfully completed:

- Wazuh Server deployed successfully
- Wazuh Manager configured and verified
- Windows Agent installed and connected
- Linux Agent installed and connected
- Endpoint registration verified
- File Integrity Monitoring (FIM) configured
- Threat Hunting dashboard verified
- MITRE ATT&CK module explored
- Configuration Assessment completed
- Security events generated and verified
